    # Nvm、Node、Pnpm

    # Nvm安装

    # 为什么使用nvm?

    不同项目需要不同node环境, 安装大量node环境太麻烦也费时间, 使用nvm可以很好解决这个问题

    nvm下载地址 (opens new window)

    安装完成后以管理员模式打开cmd输入nvm -v出现对应版本号表示安装成功

    在nvm安装目录下创建nodejs目录然后修改系统变量将NVM_SYMLINK修改为nodejs目录所在的路径xxx\nodejs用户变量和系统变量都要修改settings.txt文件的path配置也要修改

    # 配置阿里云镜像

    编辑nvm安装目录下settings.txt文件追加如下内容

    node_mirror: https://npmmirror.com/mirrors/node/
npm_mirror: https://registry.npmmirror.com/

    # 相关命令

    # nvm ls

    查看已安装的node版本

    # nvm list available

    查看可安装的node版本

    以18.20.2版本为例

    执行nvm install 18.20.2

    执行nvm use 18.20.2

    # 常用命令

    nvm install <version>		# 安装指定版本node
nvm install stable			# 安装最新稳定版node
nvm uninstall <version>		# 卸载指定版本node
nvm ls						# 显示已安装的node版本
nvm list available			# 列出所有可安装node版本
nvm use <version>			# 切换到指定版本node
nvm current					# 查看当前使用的node版本
nvm root					# 查看nvm根目录
